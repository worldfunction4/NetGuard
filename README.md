## 自顶向下的系统架构
┌─────────────────────────────────────────┐
│  展示层（Vue3 + ECharts）                │  ← 拓扑图、设备状态、报告预览
│  - 网络拓扑可视化                        │
│  - 配置版本对比（类似Git diff）           │
│  - 合规检查结果看板                      │
├─────────────────────────────────────────┤
│  应用层（FastAPI + SQLAlchemy）          │  ← RESTful API、权限管理、任务调度
│  - 设备管理CRUD                          │
│  - 基线规则引擎                          │
│  - 报告生成服务                          │
├─────────────────────────────────────────┤
│  网络层（Netmiko + SNMP + Scapy）        │  ← 和设备打交道的核心
│  - SSH/Telnet配置采集（Netmiko）         │
│  - 设备自动发现（SNMP/sysDescr）          │
│  - 网段存活扫描（Scapy）                 │
├─────────────────────────────────────────┤
│  数据层（SQLite + Redis + 文件存储）      │  ← 配置持久化
│  - 设备信息/配置版本                      │
│  - 异步任务队列（Redis）                  │
│  - 配置文件Git-like存储                   │
└─────────────────────────────────────────┘
